# 開発ログ — 2026-04-13

Kismet Enhanced UI（`http_data` 中心）の改修内容まとめ。作業ブランチは主に `master` にマージ済み。

---

## 1. Enhanced UI 一式（初期実装）

- **i18n**: `kismet_i18n.js`、`locales/en|ja/translation.json`、vendor スタブ（i18next / Chart / Papa / jsPDF 等）
- **ホワイトリスト**: `kismet_whitelist_api.js`（localStorage + タグ同期試行）、`kismet_ui_whitelist.js`（jsPanel + Tabulator）
- **未接続クライアント**: `kismet_ui_unassociated.js`（フィルタ・CSV/PDF・一括登録など）
- **信号**: `kismet_ui_signal_filter.js`、`kismet_ui_signal_monitor.js`（WebSocket / ポーリング / モック）
- **エクスポート**: `kismet_ui_export.js`
- **メイン統合**: `kismet_ui_enhanced.js`（承認列・行ハイライト・言語ペイン）
- **スタイル / ローダー**: `kismet_enhanced.css`、`kismet_enhanced_loader.js`、`index.html` 組み込み
- **検証**: ルート `test.py`（48 テスト）

## 2. Kali 実機向け修正

- **ローダー**（`kismet_enhanced_loader.js`）  
  - `_enhancedInitialized` で二重初期化防止  
  - `autoStart()` で DOM 準備後に `kismet_enhanced_run_async` 自動実行  
  - 未設定時 `localStorage` 既定言語 `ja`
- **未接続クライアント**  
  - フィルタ: `Wi-Fi Client` + `last_bssid` / `num_client_aps` 等  
  - API 主: `phy-IEEE802.11`、失敗時 `all` フォールバック  
  - 15 秒 `setInterval` 自動更新  
  - jsPanel 3/4・HTML テーブルフォールバック  
- **フォント**: `http_data/fonts/jsglyph.ttf` 空ファイル（404 抑制）

## 3. 信号フィールド名

- `kismet.common.signal.last_signal_dbm` → **`last_signal`**（実 API 合わせ）  
- 対象: `kismet_ui_unassociated.js`、`kismet_ui_signal_monitor.js`、`kismet_ui_export.js`

## 4. 未接続一覧 → 信号モニター

- Tabulator 5 はコンストラクタの `rowClick` が効かないため、**`unassocTable.on("rowClick", …)`** に変更  
- HTML フォールバック表にも行クリックでモニター起動

---

## 5. 言語・日本語 UI 揃え

- **言語設定**: 英語/日本語ラベル・説明・`windowTitle` を `t()` 化  
- **設定ダイアログ接頭辞**: `settings.dialog_title_prefix`（日本語「設定」）— `kismet.ui.settings.js` で `kismet_i18n` 利用時に反映  
- **行ハイライト表示名**: `name` はストレージ用に固定、`label` で日本語表示（`kismet.ui.js` 表示側対応）  
- **未接続画面**: CSV/PDF・フィルタ・更新表示・閉じる・エラー文などを翻訳キー化  
- **信号モニター**: モック接頭辞 `signal_monitor.mock_prefix`（`[モック] `）  
- **`kismet_i18n.js`**: 未対応言語・フォールバックを **`ja`** 寄りに変更  
- **翻訳 JSON**: `highlight.*`、`common.yes/no`、`jspdf_missing`、`select_rows_first`、`unassociated.*` 等を en/ja 同期追加
